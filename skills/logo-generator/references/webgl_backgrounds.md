# WebGL Dynamic Backgrounds

Use this file when the user wants interactive, shader-style logo presentations instead of static showcase renders.

## Available directions

### LED Matrix
- Dense glowing point grid
- Retro-digital and cybernetic feel
- Good for AI, SaaS, and infra products

### Fluid Warping
- Domain-warped gradients and motion
- Good for creative tools and fluid brands

### Fabric Wave
- Silk-like undulation
- Good for premium or luxury presentations

### Off-Center Ripple
- Corner-driven ripple motion
- Good for asymmetric or dynamic systems

### Holographic Dispersion
- Iridescent color splitting
- Good for premium tech and optical products

### Spiral Vortex
- Rotational energy and circular motion
- Good for motion-centric brands

## Usage notes

- Dark backgrounds need light logos.
- Light backgrounds need dark logos.
- Use subtle motion and high contrast.
- Favor fullscreen previews or 16:9 cards.

Use [../assets/background_library.html](../assets/background_library.html) as the visual starting point.
